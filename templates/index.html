Column Definitions
Score
Weighted priority score (0–1) combining variability amplitude (25%), reduced χ² (20%), Gaia G brightness (20%, Cat B only), number of observations (15%), galactic latitude (10%), and number of filters (10%). Category C uses brightness + galactic latitude only.
Amplitude
Peak-to-peak magnitude range (max − min) in the best-observed ZTF band. Larger values indicate stronger brightness variations. Not available for Category C (Gaia-only sources).
Nobs
Number of good observations in the best-observed ZTF band. Higher counts improve statistical confidence in variability measurements.
Gaia G
Mean apparent magnitude in the Gaia G band (330–1050 nm). Only available for sources with a Gaia counterpart (Categories B and C).
ZTF cat sep
Angular separation (arcsec) to the nearest ZTF catalog source (within 5″). Only for Category C. Sources within 2″ with consistent brightness were already removed as false positives. This is from the ZTF object catalog, not forced photometry.
ZTF cat mag
Mean magnitude of the nearest ZTF catalog source. Compare with Gaia G to assess whether the Gaia source might be the same object.
|Δmag|
Absolute magnitude difference between Gaia G and the nearest ZTF catalog source. Small values (< 2 mag) suggest the sources might be the same object with an astrometric offset.
Surveys
External catalogs with a positional match within 3″ (VizieR cone search). Checked for the top {{ n_surveyed }} candidates only: 2MASS, AllWISE, PanSTARRS DR1, SDSS DR12.

Category Descriptions

Category A
ZTF sources with no Gaia counterpart within {{ xmatch_radius }}″. Could be transients, high proper-motion objects, or ZTF artifacts.
Category B
ZTF sources matched to Gaia but not flagged as variable in Gaia DR3. Potentially new variable stars.
Category C
Gaia sources with no ZTF counterpart within {{ xmatch_radius }}″. Could be very red/faint sources below ZTF detection, sources in chip gaps, or transients that faded.

Sources brighter than {{ bright_mag_limit }} mag (in either ZTF or Gaia G) are excluded due to ZTF saturation. Sources with missing or invalid photometry are also removed.
